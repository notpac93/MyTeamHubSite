Three simple workflows MyTeam is built to support

Each workflow is straightforward: set it up clearly, give families one path to follow, and spend less time stitching the experience together.

Season Setup

Before registration opens

Set up your program once

Start from one organized source instead of rebuilding the same information across pages, forms, and messages.

Add the program overview, dates, locations, pricing, and expectations in a format families can scan quickly.

  • Publish the basics families ask for first
  • Keep fees and dates tied to the program page
  • Update one clear source when details change

Registration

When families are ready to act

Guide signup and payment in one flow

Give families a direct path from reviewing the program to completing the next step.

Keep registration decisions closer to the program details so staff do less explaining and families do less hunting.

  • Show registration options clearly
  • Connect payment expectations to the signup step
  • Reduce dropped handoffs between interest and completion

Weekly Operations

During the season

Manage schedule changes and updates

Keep the weekly rhythm of the program easier to follow for staff and families.

Post the schedule, share updates, and give families a consistent place to check what changed.

  • Publish schedule updates without a separate thread
  • Keep reminders tied to the program experience
  • Cut down on repeat questions about timing and location
